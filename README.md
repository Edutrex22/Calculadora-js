# Calculadora JavaScript

Uma calculadora web moderna e responsiva desenvolvida com HTML, CSS e JavaScript puro.

## Funcionalidades

- Operações básicas: adição, subtração, multiplicação e divisão
- Design responsivo que funciona em dispositivos móveis e desktop
- Interface intuitiva com feedback visual
- Suporte a teclado físico
- Histórico de operações
- Tratamento de erros (ex: divisão por zero)

## Como Usar

1. Acesse a calculadora em [GitHub Pages](https://edutrex22.github.io/Calculadora-js/)
2. Use os botões na tela ou o teclado para realizar os cálculos
3. Teclas suportadas:
   - Números: 0-9
   - Operadores: +, -, *, /
   - Enter: =
   - Esc: Limpar
   - Backspace: Apagar último dígito

## Tecnologias Utilizadas

- HTML5
- CSS3 (Grid e Flexbox)
- JavaScript Vanilla
